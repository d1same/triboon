Live status from the Triboon server.

Usenet providers

Add one or more NNTP providers — extras act as failover when articles are missing. Each has its own connection limit. Credentials are encrypted at rest and never sent to clients.

Indexers

Newznab-compatible indexers — searches fan out to all of them in parallel and results are merged.

Max release size

Auto sets no hard limit — ranking simply prefers sizes that start fast, while every release stays in Sources. Manual HIDES releases above your GB caps from Sources and auto-play entirely.

Scoring

Releases are ranked with built-in TRaSH-Guides-style weights tuned for instant streaming — leaving everything empty keeps that recommended default. Your entries layer on top: group names match the release's -GROUP suffix and override the built-in tiers; keywords add their score to any release whose name contains them.

TMDB metadata

Server-side TMDB key — clients browse through the server proxy and never see it.

Subtitles (Wyzie Subs)

Online subtitles for releases that only carry image-based (BluRay PGS) subs — which can never become text captions. Claim a free API key at store.wyzie.io/redeem and paste it here — no account login, no download quota. Results are cached per title.
